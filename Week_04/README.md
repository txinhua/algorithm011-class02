学习笔记

本周主要学习的内容为:
1. 深度优先搜索(DFS)
2. 广度优先搜索(BFS)
3. 贪心算法
4. 二分查找

##### **深度优先**
```
可以理解为,先父子后兄弟,沿着某一个儿子的分支统计下去,统计完成后再统计下一个儿子的数据,这样逐个报数.
可以完成家族中成员统计的分支.为了实现这种算法结构我们需要FILO 所以要么写递归利用系统的方法栈实现stack先入后出的功能,要们自己使用stack维护.
```
##### **广度优先**
```
即,FIFO 使用queue来维护子节点的出入顺序.保证其为FIFO.
```
